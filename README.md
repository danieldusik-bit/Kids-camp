# Детский лагерь 2026 — Портал регистрации

Веб-приложение для регистрации детей в летний лагерь 2026. Включает публичную форму регистрации и административную панель для управления заявками.

## Технологии

- **Next.js 14** (App Router)
- **Tailwind CSS** — стилизация
- **Prisma + SQLite** — база данных
- **NextAuth.js** — аутентификация
- **bcryptjs** — хеширование паролей

## Установка и запуск

```bash
# 1. Установить зависимости
npm install

# 2. Создать базу данных и применить миграции
npx prisma migrate dev --name init

# 3. Заполнить базу начальными данными (создаёт суперадмина)
npx prisma db seed

# 4. Запустить приложение
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Вход в админ-панель

- **URL:** http://localhost:3000/admin
- **Email:** admin@camp.com
- **Пароль:** Admin1234!

## Разделы

### Публичная форма регистрации (`/register`)

Форма в стиле Google Forms для родителей. Содержит поля для информации о ребёнке, родителе, контактных данных и медицинской информации. После отправки данные сохраняются в базу.

### Административная панель (`/admin`)

- **Дашборд** — статистика по заявкам и последние регистрации
- **Заявки** — полная таблица всех заявок с поиском, фильтрацией, экспортом в CSV и детальным просмотром
- **Пользователи** (только для суперадмина) — управление учётными записями команды

### Роли пользователей

- **Суперадмин** — полный доступ: управление пользователями, удаление заявок
- **Менеджер** — просмотр, изменение статуса заявок, добавление заметок
- **Просмотр** — только чтение заявок
